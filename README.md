# To-Do
Programa de To-dos feito no C++.

#### Projeto de um mini aplicativo de tarefas, desenvolvido totalmente por mim.

### Funções:
- Funções online
- Sistema de Login 
- Criar, editar e apagar to-dos
    Registro de quanto foi criado e foi alterado
- Historico de To-dos
- Sistema de ADM
    podendo criar e finalizar todos dos outros
